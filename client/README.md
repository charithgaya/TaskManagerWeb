# Task Manager Web App (Frontend)

## 🚀 Live Demo
https://task-manager-web-j7pt.vercel.app

## 📌 Description
This is the frontend of the Task Manager Web Application built using Next.js.  
It allows users to manage tasks, authenticate, and reset passwords.

## 🛠 Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## ✨ Features
- User Authentication (Login/Register)
- Task Management (Create, Update, Delete)
- Dark Mode
- Forgot & Reset Password

## ⚙️ Setup Instructions

```bash
git clone https://github.com/charithgaya/task-manager-frontend.git
cd task-manager-frontend

npm install
npm run dev
